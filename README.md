# CA1 - Advanced Programming Techniques

**Institution:** Dublin Business School (DBS)  
**Program:** MSc in Cyber Security  
**Module:** Advanced Programming Techniques  

## Overview

This repository contains the source code for the Continuous Assessment 1 (CA-1). The project is divided into four directories, each corresponding to a specific mini-project or "Question" required by the assessment brief.

## Repository Structure

### [Que1](./Que1) 
* **Language:** C#
* **Description:** A simulation of a digital Contact book. This application allows users to manage and check a list of contacts.

### [Que2](./Que2) 
* **Language:** C#
* **Description:** A utility tool designed to analyze file extensions. The user inputs a specific file extension, and the system retrieves and displays detailed information regarding that file type.

### [Que3](./Que3)
* **Language:** Python
* **Description:** A network programming implementation using a Client-Server architecture. This project simulates a management application for a university Master's program, handling requests between the client interface and the server backend.

### [Que4](./Que4) 
* **Language:** Python
* **Description:** This system extracts data from target websites to collect hotel rooms information. The collected data is processed and exported into a structured CSV file for further analysis.
